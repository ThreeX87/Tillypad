Связка Продукт/Принтер/Отчет создается таким образом:
1. У отчета прописывается в скрипте id группы продуктов, которые должны печататься
2. У этого же отчета в комментарии прописывается этот же id
3. У группы продуктов в комментарии прописывается id принтера, на котором должна происходить печать

Таким образом, если в одной накладной есть продукты из разных групп, то печать будет разделенной, т.е. на каждый принтер отправится на печать только те продукты, которые должны быть на нем распечатаны
